제주도는 어떤 곳인가요?

한반도 남서쪽에 위치한 제주도는 한국 최대의 섬이자 독특한 화산 지형, 온화한 기후, 독자적인 역사와 삼다(돌, 바람, 여자) 문화를 간직하고 있는 최고의 휴양지입니다.

유네스코 세계유산

제주 화산섬과 용암동굴은 유네스코가 지정한 대한민국 최초의 세계자연유산으로 가치를 인정받고 있습니다.

사계절 따뜻한 기후

연중 온화한 난대성 기후로, 추운 겨울에도 이국적인 상록수와 귤나무, 동백꽃을 만나볼 수 있습니다.

다채로운 액티비티

에메랄드빛 해변에서 서핑, 패들보드를 즐기고 오름 트레킹과 올레길 걷기로 힐링의 시간을 가질 수 있습니다.

GitHub Pages 배포 가이드

이 HTML 소스 코드를 사용하여 나만의 멋진 포트폴리오 사이트를 깃허브에 무료로 배포해 보세요! 아래 단계를 따라 클릭 몇 번이면 끝납니다.

1. GitHub 로그인 후 우측 상단의 + New 버튼을 누릅니다.

2. Repository name에 저장소 이름(예: my-jeju-page)을 입력합니다.

3. Public을 선택하고, 하단의 Create repository를 클릭합니다.

1. 생성된 빈 리포지토리 화면에서 uploading an existing file 링크를 클릭합니다.

2. 현재 이 페이지 소스 코드가 저장된 index.html 파일을 마우스로 드래그하여 업로드합니다.

3. 화면 하단의 녹색 Commit changes 버튼을 누릅니다.

1. 리포지토리 상단의 메뉴 탭에서 ⚙️ Settings를 클릭합니다.

2. 왼쪽 사이드바에서 Pages 탭을 선택합니다.

3. Build and deployment의 Branch 항목을 None에서 main(또는 master)으로 변경하고 우측의 Save를 클릭합니다.

4. 1~2분 정도 기다린 후, 상단에 생성된 "Your site is live at..." 주소로 이동하면 전 세계 누구나 접속할 수 있는 나만의 제주도 웹사이트가 생성됩니다! 🎉
